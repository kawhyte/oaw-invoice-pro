# How to Combine Invoices for the Same Client

**When to use:** Combine invoices when you want to bill a client for **multiple projects on a single invoice**, but still show each project's work separately.

---

## Step-by-Step

### 1. **Go to Clients**
   - Click **Clients** in the menu
   - Click on the client's name (e.g., "Edwin Jones")

### 2. **Check for Draft Invoices**
   - You'll see all their projects listed
   - Make sure each project has a **draft invoice** (status = "Draft")
   - If a project doesn't have a draft, create one first in the project page

### 3. **Click "+ Combined Invoice"**
   - Only shows if the client has **2+ projects**
   - A form will open

### 4. **Pick Which Projects to Combine**
   - Check the boxes next to the projects you want on **one invoice**
   - You can combine 2 projects, 3 projects, or all of them
   - Their draft invoices will be pulled in automatically
   - Their items will appear grouped by project name (you'll see subtotals for each)

### 5. **Set Invoice Details** (if needed)
   - **Discount** — apply to the whole combined invoice
   - **GCT (15%)** — check if needed
   - **Additions** — extra charges
   - **Due Date** — when payment is due
   - **Notes** — any message for the client
   - Leave everything blank if you don't need it

### 6. **Click Create**
   - One combined invoice is created
   - The individual project drafts are **automatically deleted** (so you don't double-bill)
   - The new invoice shows a **"Combined"** badge
   - On the invoice PDF, each project's items are clearly separated with subtotals

---

## Example

**Client: Edwin Jones**
- Project A (Pool House) → Draft invoice with 3 line items
- Project B (Gates) → Draft invoice with 2 line items  
- Project C (Gazebo) → Draft invoice with 1 line item

**Combine A + B:**
- One invoice is created with both projects' items
- It shows:
  - **Pool House** section with 3 items + subtotal
  - **Gates** section with 2 items + subtotal
  - **Total** at the bottom

Project C's draft stays as-is (separate invoice).

---

## If You Change Your Mind

**Undo a combine (Separate):**
- Open the combined invoice (in the invoice detail page)
- Click **"Separate"** button
- The invoice is split back into individual project drafts
- Each gets its **original invoice number** back
- Now you can edit or delete them separately

---

**That's it!** Keep it simple — combine when it makes sense for your workflow, separate if you need to fix something.
